TASK 4 — SEQUENCE DIAGRAMS
---------------------------
What's needed:
  Sequence diagrams for the key system flows.

How to complete it:
  Recommended 5 flows to cover:

  1. Login
     Client → POST /api/auth/login → validate credentials → return JWT

  2. Submit Assignment
     Client → check attendance >= 80% → POST /api/assignment/submit
     → store Submission → return confirmation

  3. Mark Attendance
     Instructor → POST /api/attendance → create AttendanceSession
     + AttendanceRecord per student

  4. Grade Release
     Instructor grades last item → system checks all items graded
     → sets ModuleProgress.FinalGrade → creates Notification → sends email

  5. Deadline Reminder (Background Service)
     DeadlineReminderService timer fires → queries assignments due in 48hrs
     → for each: create Notification + send email (skips duplicates)

  Use PlantUML (free, text-based) or draw.io.
  Save each as a separate file:
    docs/diagrams/seq-login.png
    docs/diagrams/seq-submit-assignment.png
    docs/diagrams/seq-mark-attendance.png
    docs/diagrams/seq-grade-release.png
    docs/diagrams/seq-deadline-reminder.png
